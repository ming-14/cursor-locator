# 鼠标小圆圈

## 分层架构总览

```
鼠标小圆圈/                     (项目根目录)
│
├── run.py                     [入口层]  启动脚本（版本检查 / 单实例 / 控制台最小化）
│
├── src/                       [主源码包]
│   ├── main.py                [入口层]  入口编排 + 后台线程 + Tkinter 主循环
│   │
│   ├── ui/                    ─── UI 层 ───
│   │   └── settings_panel.py  Tkinter 控制面板（消息驱动更新）
│   │
│   ├── service/               ─── 服务层 ───
│   │   ├── ring_worker.py     环线窗口线程（窗口 / 渲染 / 鼠标跟踪 / 匀速插值）
│   │   └── tray_worker.py     托盘图标线程（图标 / 菜单 / 快捷键注册）
│   │
│   ├── core/                  ─── 核心层 ───
│   │   ├── config.py          线程安全配置管理（JSON 持久化）
│   │   ├── rendering.py       圆环渲染（alpha 掩码 + BGRA 合成，纯函数）
│   │   ├── pixel_color.py     屏幕取色 + 5 种互补色算法
│   │   ├── hotkey_manager.py  多策略热键管理（RegisterHotKey / 钩子 / 轮询）
│   │   └── cpu_monitor.py     进程 CPU 占用率监控
│   │
│   └── infra/                 ─── 基础设施层 ───
│       ├── win32_api.py       Win32 常量 / 结构体 / API 声明（纯声明）
│       └── logger.py          日志模块（Loguru 可选，未安装时自动降级为静默）
│
├── test/                      单元测试
│
├── doc/
│   ├── 架构规范.md             此文档
│   ├── 编码规范.md             编码与命名约定
│   └── 日志系统设计规范（Python程序，基于 Loguru）.md
```

## 启动流程

```
run.py 启动流:
  ├── Python 版本检查（< 3.8 弹窗退出）
  ├── 单实例检测（Named Mutex）
  │   └── 已有实例 → FindWindow → SetForeground → exit(0)
  ├── 控制台窗口最小化
  └── from src.main import main → main()

main() 启动流:
  ├── init_logging()           ← Loguru 可选，未安装则静默
  ├── Config()                 ← 单例配置
  ├── queue.Queue()            ← 跨线程消息通道
  ├── threading.Thread(_run_backend)  ← 后台 Win32 线程
  ├── tk.Tk() + SettingsPanel  ← UI 线程
  ├── poll_queue()             ← 轮询消息队列（100ms）
  └── root.mainloop()
```

## 层级职责

### 入口层 (Entry) — `run.py` + `src/main.py`

负责进程级生命周期：版本检查、单实例互斥、创建全局单例（`Config`、消息队列）、启动后台线程、初始化 Tkinter 主循环。

### UI 层 (User Interface) — `src/ui/`

单一 Tkinter 窗口，负责控制面板的界面呈现与用户交互。不直接操作环线或托盘；所有状态更新通过消息队列驱动。

| 文件 | 职责 |
|---|---|
| `settings_panel.py` | `SettingsPanel`：圆环参数 / 颜色选项 / 性能设置 / 实时状态 / CPU 监控 / 快捷键捕获 / 显示器帧率 |

```
SettingsPanel 消息驱动:
  handle_message(msg)
  ├── MSG_COLOR        → 更新鼠标像素色块 + 圆环颜色色块
  ├── MSG_ON_RING      → 更新悬停状态显示
  ├── MSG_HOTKEY_LVL   → 更新快捷键注册提示
  └── MSG_REFRESH_RATE → 更新显示器帧率显示
```

### 服务层 (Service) — `src/service/`

两个长生命周期服务，运行在后台 Win32 消息循环线程中。每个服务拥有自己的 Win32 窗口和消息泵。

| 文件 | 职责 |
|---|---|
| `ring_worker.py` | `MouseRingWindow`：环线窗口创建/销毁/渲染；低层鼠标钩子管理；双跟踪模式切换；匀速平滑插值；显示器刷新率检测 |
| `tray_worker.py` | 托盘通知窗口：图标添加/删除/右键菜单；快捷键重接入 |

```
ring_worker.py 内部架构:
  MouseRingWindow
  ├── 窗口初始: RegisterClass → CreateWindowEx → DIBSection
  │   └── SetWindowDisplayAffinity(WDA_EXCLUDEFROMCAPTURE)
  ├── 鼠标跟踪:
  │   ├── hook 模式: SetWindowsHookEx(WH_MOUSE_LL) → _on_hook_move()
  │   └── timer 模式: SetTimer → _on_timer()
  │       ├── 阶段1: 按跟踪频率采样鼠标位置，触发平滑段
  │       ├── 阶段2: 匀速线性插值（_smooth_from → _smooth_to）
  │       └── 阶段3: 渲染帧
  ├── 定时器参数:
  │   ├── auto 模式: 间隔 = 1000 / 显示器刷新率 × 倍率
  │   └── custom 模式: 用户自定义间隔
  ├── 渲染循环: _on_tick() → 取色 → 碰撞检测 → _update_frame()
  └── 消息驱动: 面板可见时向 mq 投递 MSG_COLOR / MSG_ON_RING / MSG_REFRESH_RATE

tray_worker.py 内部架构:
  init_tray()
  ├── RegisterClass → CreateWindowEx (隐藏窗口)
  ├── Shell_NotifyIcon(NIM_ADD)
  ├── register_initial_hotkey()
  └── GetMessage loop
  _tray_wnd_proc:
  ├── WM_HOTKEY        → MSG_TOGGLE
  ├── WM_TRAY_CALLBACK → 右键菜单 / 左键切换
  ├── WM_RELOAD_HOTKEY → _reload_hotkey_internal()
  └── WM_COMMAND       → "显示设置" / "退出"
```

### 核心层 (Core) — `src/core/`

无状态或极少状态的纯逻辑模块，不依赖 UI 或窗口系统。

| 文件 | 职责 | 依赖 |
|---|---|---|
| `config.py` | 线程安全读写 + JSON 持久化 | 无 |
| `rendering.py` | `build_alpha_mask()` + `render_ring_from_mask()`（纯函数，抗锯齿 + premultiplied alpha） | 仅 `math` |
| `pixel_color.py` | `get_pixel_color()` + `compute_complement()`（5 种算法：简单反色 / HSL 旋转 / 感知亮度 / 黑白 / 灰度反色） | Win32 API |
| `hotkey_manager.py` | `HotkeyManager` 三策略快捷键注册 + 冲突检测 + 安全提示 | Win32 API |
| `cpu_monitor.py` | `CPUMonitor` 进程 CPU 占用率计算（GetProcessTimes） | Win32 API |

### 基础设施层 (Infrastructure) — `src/infra/`

| 文件 | 职责 |
|---|---|
| `win32_api.py` | 所有 Win32 常量 / 结构体 / API 加载 + `argtypes` 声明 / 回调类型 / 消息类型常量（纯声明，零逻辑） |
| `logger.py` | 基于 Loguru 的日志初始化；**Loguru 为可选依赖**，未安装时自动降级为 `_NullLogger`（静默空操作），程序正常运行不受影响 |

**依赖规则**：
- 箭头方向 = 依赖方向（被依赖者在下方）
- `core/config.py`、`infra/win32_api.py`、`core/rendering.py` **零依赖**
- 上层可以依赖下层，反之不行
- 同层模块间允许有限依赖（如 service → core）
- 统一使用 `from src.xxx import ...` 绝对导入

## 线程模型

```
┌───────────────────────────────────────────────────────┐
│ 主线程 (Tkinter UI 线程)                                │
│  Thread ID: MainThread                                 │
│  职责:                                                 │
│    · SettingsPanel UI 响应                              │
│    · poll_queue() 每 100ms 轮询消息队列                │
│    · 收到消息后调用 settings.handle_message()           │
│    · 快捷键捕获轮询 (30ms)                              │
│    · CPU 监控轮询 (1s)                                 │
└───────────────────────┬───────────────────────────────┘
                        │  queue.Queue
                        ▼
┌───────────────────────────────────────────────────────┐
│ 后台线程 (Win32 消息循环线程)                            │
│  Thread ID: Thread-1 (_run_backend)                    │
│  职责:                                                 │
│    · MouseRingWindow 环线窗口管理                       │
│    · 低层鼠标钩子 (WH_MOUSE_LL) 安装/卸载               │
│    · 定时器跟踪 → _on_timer()（采样 → 插值 → 渲染）    │
│    · 钩子跟踪   → _on_hook_move()                      │
│    · 取色 + 碰撞检测 + 渲染帧                           │
│    · 面板可见时投递 UI 更新消息                          │
│    · 托盘图标管理 + 右键菜单                            │
│    · 快捷键监听 (WM_HOTKEY)                            │
│    · 显示器刷新率检测 (EnumDisplaySettingsW)            │
│    · GetMessage 消息循环                               │
└───────────────────────────────────────────────────────┘
```

### 线程安全规则

1. **Config**：内部 `Lock`，读写安全
2. **queue.Queue**：唯一跨线程数据通道
3. **Win32 窗口句柄**：仅在创建线程中操作（hwnd 线程亲缘性）
4. **Tkinter 控件**：仅在主线程操作

## 消息类型定义

定义在 `src/infra/win32_api.py`，集中维护。

```python
# 服务层 ──队列──→ UI 层

MSG_COLOR       = 'color'         # 颜色更新 → {type, pixel, complement}
MSG_ON_RING     = 'on_ring'       # 悬停状态变化 → {type, value}
MSG_HOTKEY_LVL  = 'hotkey_level'  # 快捷键注册状态 → {type, level, msg}
MSG_REFRESH_RATE = 'refresh_rate' # 显示器刷新率 → {type, freq}
MSG_SHOW        = 'show_settings' # 显示控制面板
MSG_TOGGLE      = 'toggle_settings'  # 切换面板显隐
MSG_EXIT        = 'exit'          # 退出程序
```

## 运行环境要求

- **操作系统**：Windows 10 / 11
